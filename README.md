# Кухня в Дом

Профессиональный лендинг для кухни и мебели на заказ в Анапе.

## Запуск

```bash
npm install
npm run dev
npm run build
```

## Подключение заявок

Форма отправляет JSON на серверный webhook. Это позволяет безопасно направлять заявки в Make, n8n, CRM или другой серверный обработчик без хранения секретов в браузере.

Создайте файл `.env` на основе настроек ниже:

```bash
VITE_LEADS_WEBHOOK_URL=https://your-server.example/webhook
VITE_LEADS_WEBHOOK_TOKEN=
VITE_YM_ID=
```

`VITE_LEADS_WEBHOOK_URL` обязателен для работы отправки. Адрес должен принимать POST-запрос в формате JSON и отвечать успешным HTTP-статусом 2xx. Не размещайте в клиентском коде токен Telegram-бота, пароль CRM или другие секреты.

## Что приходит в webhook

Имя, телефон, ответы квиза, ориентир бюджета, комментарий, удобное время звонка, адрес страницы, referrer и время отправки.

## Перед публикацией

1. Подключите webhook для заявок и проверьте тестовую отправку.
2. Добавьте номер счетчика Яндекс Метрики в `VITE_YM_ID`.
3. Замените карточки отзывов на подтвержденные отзывы и добавьте ссылки на источники.
4. Укажите полное наименование владельца бизнеса и ИНН на странице `public/privacy.html`, если они применимы.
5. После подключения домена добавьте абсолютный URL sitemap в `public/robots.txt`.
